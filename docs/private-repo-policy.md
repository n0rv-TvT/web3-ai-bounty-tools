# Private Repo Policy

This repository is for private Web3/AI bug bounty research continuity, backup, and organization. It is not a public product repo.

Use it to preserve reusable framework scripts, prompts, templates, sanitized evidence summaries, status decisions, and target research history.

Do not use it to store secrets, raw platform messages, cookies, real user data, private keys, seed phrases, unsanitized logs, or large generated build artifacts.

Before committing, check:

1. Is this useful for resuming work later?
2. Is it sanitized?
3. Does it avoid secrets and private platform/session data?
4. Is it a clean summary rather than raw noise?
5. Does every finding have a status and next action?
