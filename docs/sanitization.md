# Sanitization

This public repo must contain only reusable, sanitized material.

Never commit:

- `.env` files
- private keys or seed phrases
- wallet files
- RPC URLs
- API tokens
- session cookies
- private platform messages
- real user data
- unsanitized target reports
- raw exploit logs
- private target repositories or private program data

Safe to commit:

- reusable skills and prompts
- framework scripts and templates
- toy examples
- sanitized report drafts
- sanitized evidence summaries
- public-safe docs and workflows

When in doubt, commit a short summary instead of the raw artifact.
