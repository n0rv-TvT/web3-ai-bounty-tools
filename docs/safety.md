# Safety

Use this toolkit only for authorized Web3 security research, bug bounty programs, private labs, CTFs, and owned code.

Do not use real private keys, seed phrases, user funds, session cookies, or platform tokens in examples or committed files.

Prefer local tests and read-only analysis. Do not broadcast transactions unless you have explicit authorization outside this repo.

Scanner output and AI-generated findings are leads only. A finding needs source confirmation, reachability, concrete impact, and a working PoC before report drafting.
