# Web3-AI Bug Bounty Research Framework

> Legacy private-workflow note. This repository is now intended to publish sanitized reusable skills, prompts, scripts, templates, and workflows. Do not commit real private target data, secrets, raw logs, cookies, RPC URLs, or unsanitized reports here.

Private research framework for PoC-first Web3/AI bug bounty validation.

This repo is not a public product. It is my private research operating system for:

- saving reusable prompts
- organizing target research
- tracking finding status
- generating and validating local PoCs
- storing evidence packages
- avoiding repeated killed/duplicate/N/A ideas
- preserving sanitized report drafts and platform responses
- making old work easy to resume later

Core workflow:

Lead → Hypothesis → Code Review → PoC → Execution Gate → Evidence Package → Report Gate → Submit / Kill / Archive

Final rule:

Commit clean summaries, decisions, prompts, scripts, templates, and sanitized evidence.
Ignore secrets, raw logs, noisy generated artifacts, unsanitized platform material, and messy temporary notes.
