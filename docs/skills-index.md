# Skills Index

## Imported Opencode skills

Sanitized Opencode skill folders are included under `skills/` for reusable workflow continuity. The largest imported skill is `web3-ai-bounty`, which includes reusable docs, scripts, schemas, examples, and toy eval fixtures. Benchmark corpora, raw sources, generated outputs, and scoring artifacts were intentionally excluded.

## `web3-bounty-xray`

Map a Web3 target's contracts, roles, entry points, asset flows, trust boundaries, and high-value attack surfaces.

## `poc-first-validator`

Decide whether a lead should be proven, killed, downgraded, or considered report-ready.

## `foundry-poc-writer`

Turn a hypothesis into a minimal Foundry exploit test and control test plan.

## `evidence-gate-reviewer`

Check whether a finding's code refs, PoC, impact, and duplicate notes are complete enough for report drafting.

## `report-readiness-checker`

Run the strict 7-question report gate.

## `duplicate-na-risk-checker`

Identify duplicate, known-risk, intended-behavior, and likely N/A issues.

## `ai-agent-web3-threat-modeler`

Map AI-agent read/tool/signing boundaries and harmful prompt-injection-to-capability chains.
