# Prompts

Reusable public-safe prompt and agent command material.

- `opencode-commands/` contains sanitized Opencode slash-command prompts.
- `opencode-agents/` contains sanitized Opencode agent definitions.
- `opencode-agents-legacy/` contains legacy single-agent definitions if present.

Do not commit target-specific prompts containing private program data, cookies, RPC URLs, tokens, or unsanitized report details.
