# Contributing

This repository is for reusable, sanitized Web3/AI security research skills, prompts, scripts, templates, and toy examples.

## Pull request checklist

- [ ] No private keys, seed phrases, `.env` files, cookies, tokens, RPC URLs, or credentials.
- [ ] No real private bounty target data or unsanitized platform messages.
- [ ] No raw exploit logs from live systems.
- [ ] Examples are synthetic, public, or clearly safe to redistribute.
- [ ] New skills include a clear description, when to use, inputs expected, output format, safety rules, and example prompt.
- [ ] Scripts include a short usage note or are referenced from `docs/script-index.md`.

## Good contributions

- PoC-first validation prompts.
- Evidence-gate improvements.
- Report-readiness checks.
- Foundry PoC templates and toy fixtures.
- AI-agent/Web3 signing or tool-use threat-model prompts.
- Sanitized bug-class playbooks.
- Script hardening, tests, and schema improvements.

## Avoid

- Scanner-only finding dumps.
- Private target notes.
- Real exploit artifacts from live systems.
- Marketing claims that imply guaranteed findings or autonomous hacking.

Keep the project practical, evidence-focused, and safe for authorized research.
